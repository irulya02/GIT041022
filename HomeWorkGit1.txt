First level: 1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
1. зайти в созданный репозиторий 
2. нажать зеленую кнопку Code
3. выбрать SSH ключ и скопировать его.
4. локально ввести команду git clone ключ 



Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
1. через команду mkdir создать директорию
2. создаем или копируем файл 
3. вводим команду git init
4. вводим команду git status
5. добавляем файлы командой git add 
6. командой git commit -m "add something" создаем коммит
7. на github  создаем новый репозиторий с именем как на локальном каталоге (без README file)
8. Добавляем предложенные github команды 

git remote add origin git@github.com:irulya02/1234567.git
git branch -M main
git push -u origin main

  

Как получить на локальный компьютер изменения с github.com
Через команду git pull

Какую информацию нам дают команда :
git status - показывает неотслеживаемые файлы, которые не добавлены в коммит
git log - показывает коммиты
git remote -v - связь локального каталога с другим репозиторием
